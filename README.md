# VP-LO Kraków — nowoczesna wersja strony

Projekt przygotowany w React + Vite, gotowy do otwarcia w IntelliJ IDEA.

## Uruchomienie

```bash
npm install
npm run dev
```

## Build produkcyjny

```bash
npm run build
npm run preview
```

## Co zawiera

- responsywną stronę główną,
- zachowane główne zakładki i podstrony,
- nowoczesne menu desktop/mobile,
- gotowe miejsca na treści, dokumenty, galerię i formularze.

## Gdzie edytować

Najważniejszy plik:

- `src/App.jsx` — struktura podstron i treści,
- `src/styles.css` — stylowanie.
